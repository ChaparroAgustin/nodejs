# Proyecto NOC 

El objetivo es crear una serie de tareas usando Arquitectura Limpia con TypeScript.

# dev 

1. clonethe file env.template to .env

2. config enviroment variables
```
PORT = 3000

MAILER_SERVICE=
MAILER_EMAIL = 
MAILER_SECRET_KEY = 

PROD =false
```

3. execute ``npm install``

4. execute ``npm run dev ``
